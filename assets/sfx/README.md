# SFX — attribution

All sound effects here are sourced from Kenney's CC0 asset packs and renamed to
match the logical slots consumed by `AudioManager`.

| File              | Source pack                                             | Original file                 |
|-------------------|---------------------------------------------------------|-------------------------------|
| `ui_click.ogg`    | [Kenney UI Audio](https://kenney.nl/assets/ui-audio)    | `click3.ogg`                  |
| `move.ogg`        | [Kenney UI Audio](https://kenney.nl/assets/ui-audio)    | `switch3.ogg`                 |
| `merge.ogg`       | [Kenney Impact Sounds](https://kenney.nl/assets/impact-sounds) | `impactSoft_medium_001.ogg` |
| `combo.ogg`       | [Kenney Digital Audio](https://kenney.nl/assets/digital-audio) | `zapThreeToneUp.ogg`      |
| `win.ogg`         | [Kenney Digital Audio](https://kenney.nl/assets/digital-audio) | `powerUp5.ogg`            |
| `game_over.ogg`   | [Kenney Digital Audio](https://kenney.nl/assets/digital-audio) | `zapThreeToneDown.ogg`    |
| `achievement.ogg` | [Kenney Digital Audio](https://kenney.nl/assets/digital-audio) | `threeTone1.ogg`          |

All Kenney assets are licensed **CC0 1.0 Universal (Public Domain)**. Attribution
is not legally required but noted here as courtesy.

To refresh or replace an asset, drop a new `.ogg` into this folder with the same
filename (`AudioManager.STREAM_PATHS` resolves by key).
